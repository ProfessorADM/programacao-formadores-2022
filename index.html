O HTML permite o acesso de websites e a inserção de conteúdos, como imagens e vídeos.

Desse modo, outra plataforma conhecida é o GITHUB, que é muito utilizado por desenvolvedores,pois permite uma hospedagem prática de código-fonte e arquivos em nuvem. O GitHub na era digital é uma ferramenta de extrema importância para engenheiros de software, com uma gama de popularidade expressiva. No cenário atual, ele alcança mais de 25 milhões de usuários. Dessa forma, é possível perceber que no momento contemporâneo muitos profissionais estão buscando um aprofundamento do GitHub com o objetivo de ampliar seus campos de trabalho. Neste sentido, o GitHub é uma forma de trabalho baseado em nuvem que hospeda é um sistema de controle de versão (VCS) chamado Git. Ele oferece a oportunidade dos desenvolvedores trabalharem em conjunto em uma ajuda mútua que permite que façam mudanças em projetos compartilhados enquanto mantêm um registro detalhado do seu progresso. O que podemos entender por linguagem HTML? A Linguagem HTML, pode ser entendida como uma Marcação de Hipertexto. Ou seja, é uma linguagem de computador que faz parte da maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento.O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos. Entenda o que é CSS? O CSS é a linguagem usualmente utilizada do qual podemos entender por Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma forma de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, entre outros. De grosso modo, na prática funciona como uma camada de personalização ao conteúdo perceptível. As formas de linguagem mencionadas foram idealizadas para serem funcionais e, por serem de marcação, precisaram ser aperfeiçoadas e muitas vezes modificadas. o CSS foi criado porque o HTML não era capaz de reconhecer todas as necessidades que um programador tinha para criar padrões de formatação em possíveis páginas. Desse modo, O CSS veio como uma boa solução para aprimorar o conjunto estrutural dos sites. Por meio dele, é possível separar formatos de conteúdos, bem como definir como serão exibidos em cada página.
